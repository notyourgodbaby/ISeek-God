```python
from ascension import AscensionEngine, TokenSnapshot
from datetime import datetime, timedelta

# Create engine
engine = AscensionEngine()

# Add token snapshots (simulating price history)
now = datetime.utcnow()
for i in range(10):
    snapshot = TokenSnapshot(
        timestamp=now - timedelta(days=10-i),
        market_cap=10000.0 * (1.05 ** i),
        holder_count=100 + i * 10,
        trading_volume=1000.0 + i * 100,
        price=0.001 * (1.05 ** i),
    )
    engine.add_snapshot("MYTOKEN", snapshot)

# Analyze
metrics = engine.analyze_token("MYTOKEN")
print(f"Ascension Index: {metrics.ascension_index:.2f}/100")
print(f"Divine Alignment: {metrics.divine_alignment.value}")
print(f"Safety Score: {metrics.safety_score:.2f}/100")

# Search by alignment
from ascension import DivineAlignment
enlightened_tokens = engine.search_by_alignment(DivineAlignment.ENLIGHTENED)
print(f"Found {len(enlightened_tokens)} enlightened tokens")
```

## Core Calculations

### The Math Behind Ascension Index

```
Total Growth = (Latest Market Cap / First Market Cap - 1) × 100
├─ Capped at 300% (prevents outliers from dominating)
└─ Growth Score = min(Total Growth / 3, 100)

Market Cap Consistency = Standard Deviation of all market caps
├─ Lower volatility = more consistent
└─ Consistency Score = max(0, 100 - std_dev × 50)

Ascension Index = (Growth Score × 0.6) + (Consistency × 0.4)
└─ 60% weight on growth, 40% weight on consistency
```

**Why 60/40?** Growth without consistency is luck. Consistency without growth 
is stagnation. This balance identifies *sustainable* winners.
